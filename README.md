# 🚀 UI Component Builder

✨ A dynamic React-based UI builder that allows users to create, customize, preview, and generate reusable React components in real-time.

---

## 🌐 Live Demo

https://ui-component-builder.vercel.app

---

![UI Component Builder](./screenshot01.png)

---

## ✨ Features

✔️ Select different UI components (Button, Card, Alert, Input, Toggle, Modal)
✔️ Customize component properties dynamically
✔️ 🎨 Live preview of components
✔️ 🧾 Auto-generate JSX code
✔️ 📋 Copy code to clipboard
✔️ ⬇️ Download component as JSX file
✔️ 🌙 Dark mode support
✔️ 🔄 Reset component to default state

---

## 🛠️ Tech Stack

* ⚛️ React.js (useState Hook)
* 🎨 CSS
* 🌐 JavaScript (DOM APIs for copy/download)

---

```bash id="cbm4az"
src/
 ├── Components/
 │    ├── Button.jsx
 │    ├── Card.jsx
 │    ├── Alert.jsx
 │    ├── Input.jsx
 │    ├── Toggle.jsx
 │    ├── Modal.jsx
 ├── App.jsx
 ├── App.css
```

---

## ⚙️ How It Works

* User selects a component from dropdown
* Dynamic form controls update component properties
* Component preview updates in real-time
* JSX code is generated based on selected properties
* Users can copy or download the generated code
